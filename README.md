# 获取武汉 2019nCov信息

[![](https://img.shields.io/badge/pip-v0.0.5%20Wuhan2019nCoV-yellow.svg)](https://pypi.python.org/pypi/Wuhan2019nCoV)

> 算是基本满足我的基本需求，要的自取，欢迎PR~

## 安装

```
sudo pip install Wuhan2019nCoV
```

如果你是Mac OS，请通过Ruby安装`terminal-notifier`来接收通知:

```
sudo gem install terminal-notifier
```

## 使用

```
whncov
```
